# Buddhist Term Glossary & Known ASR Mishearings

## Core terms likely to appear in teachings

bodhicitta, bodhisattva(s), Buddha, buddhahood, Dharma, Sangha,
paramita(s), samsara, nirvana, sentient beings, karma, merit,
compassion, loving-kindness, emptiness (shunyata), single-pointedness,
Mahayana, Hinayana, Theravada, Vajrayana, sutra, tantra, mantra,
lama, geshe, rinpoche, khenpo, tulku, ngondro, lamrim, lojong,
refuge, precepts, vows, mindfulness, awareness, meditation, retreat,
dedication, aspiration, prostration, offering, mandala, stupa

## Enumerated lists (use for fixing ordinals and garbled members)

**Six perfections (paramitas):**
1. generosity (dana)
2. moral discipline / ethical discipline (shila)
3. patience (kshanti)
4. diligence / joyous effort (virya)
5. meditative concentration (dhyana)
6. wisdom (prajna)

**Five paths:** accumulation, application (preparation), seeing,
meditation, no-more-learning

**Three jewels:** Buddha, Dharma, Sangha
**Three trainings:** moral discipline, concentration, wisdom
**Four noble truths:** suffering, origin, cessation, path
**Four immeasurables:** loving-kindness, compassion, joy, equanimity

## Masters' names

Shantideva, Nagarjuna, Asanga, Atisha, Tsongkhapa, Milarepa, Marpa,
Padmasambhava (Guru Rinpoche), Longchenpa, Patrul Rinpoche

## Known mishearing patterns (observed in real transcripts)

| ASR output                      | Intended                  |
|---------------------------------|---------------------------|
| booty toois / bodey chita       | bodhicitta                |
| buddha softball / Buddhist thought was / thought well | bodhisattva(s) |
| parameter / perimeter           | paramita                  |
| profession                      | perfection                |
| more discipline                 | moral discipline          |
| part (of accumulation etc.)     | path                      |
| ensured                         | in short                  |
| fought one                      | fourth one                |
| feat one                        | fifth one                 |
| little / literate meaning       | literal meaning           |
| some sara / samsara's ocean     | samsara                   |
| centered beings                 | sentient beings           |
| lam rim / lum rim               | lamrim                    |
| sangha / sanga / sun gha        | Sangha                    |

These are patterns, not a lookup table — the same term can be garbled in new
ways. Always reason phonetically: say the ASR text aloud with a Tibetan
accent and ask what dharma term it could be.
